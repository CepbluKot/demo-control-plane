Входные параметры

Для каждого батча логов
↳ LLM NODE (внутри итерации)

output: BatchAnalysis (JSON schema)
→ возвращает list[BatchAnalysis]
▼
PYTHON CODE NODE — REDUCE

return items[0] # MergedAnalysis
