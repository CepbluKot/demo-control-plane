Архитектура Dify Workflow — v2 (с загрузкой из ClickHouse)

ClickHouse → Chunk → MAP (Iteration, parallel) → REDUCE (Code, цикл)

START
Параметры инцидента
period_start: str   period_end: str
incident_info: str   alerts: str
▼
PYTHON CODE NODE — LOAD & CHUNK
Загрузка из ClickHouse + нарезка на батчи
1. keyset-пагинация по LOGS_SQLS (containers + events)
2. zone-разметка строк (context_before / incident / context_after)
3. нарезка по токеновому бюджету → list[str]
→ возвращает batches: list[str], zones_meta: list
▼
ITERATION (parallel=true) — MAP
Для каждого батча логов
↳ LLM NODE (внутри итерации)
system: MAP prompt + alerts + incident_info
user: текст батча
output: BatchAnalysis (JSON schema)
→ возвращает list[BatchAnalysis]
▼
PYTHON CODE NODE — REDUCE
Итеративный merge (цикл)
while len(items) > 1:
